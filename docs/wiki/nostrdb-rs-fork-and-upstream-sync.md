# nostrdb-rs Fork and Upstream Sync

## Upstream Sync and Version

pablof7z/nostrdb-rs is synced with damus-io/nostrdb-rs upstream (v0.10.0), retaining the Apple LMDB pthread mutex fix and iOS stack check disable. The crate is kept at version 0.8.0 to maintain semver compatibility with the nostr-ndb crate dependency requirement of nostrdb ^0.8.0. [^8835c-6]
