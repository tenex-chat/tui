# Tool Call Detail Sheet

## Tool Call Detail Sheet

Every tool call pill in chat is a tappable button with a chevron.right indicator that opens a detail sheet. The sheet displays the tool name in monospaced text, summary text, full pretty-printed JSON arguments in a scrollable monospaced block with text selection, and content fallback text if present. On iOS, the sheet uses .presentationDetents([.medium, .large]) so it defaults to half-screen. [^92975-1]
