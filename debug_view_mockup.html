  • Mode 2 - E-Tag Explorer: Find all events that reference another event
  • Mode 3 - Kind Browser: Browse events by their kind number
  • Mode 4 - Content Search: Full-text search across event content (powered by nostrdb)
  • Mode 5 - Statistics: View cache totals and breakdown by kind
