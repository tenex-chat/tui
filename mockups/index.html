Home Screen Improvements

  • No duplicate project names: Group conversations by project or show project once per row
  • Timestamps: Show relative time (2m ago, 1h ago) for each conversation
  • Collapsed empty inbox: Don't waste space when inbox has no notifications
  • Single help bar: Consolidated keyboard shortcuts instead of two rows
  • Unread indicators: Blue dots or bold text for unread conversations
  • Activity badges: Show streaming status, reply counts for active threads

Chat Screen Improvements

  • Grouped responses: Actions belonging to one response are visually connected instead of appearing as separate "Claude Code :" blocks
  • Reduced repetition: Single assistant header instead of repeating "Claude Code :" for every line
  • Visual hierarchy: Clear distinction between user messages and assistant responses using color, backgrounds, or indentation
  • Compact actions: Tool executions shown as compact badges/lines rather than full blocks with excessive whitespace
  • Status indicators: Success/failure states shown with color coding and icons
  • Better input area: Cleaner layout with context (branch, user) and keyboard hints organized
